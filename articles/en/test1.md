# 简介

该项目是我在大学时，用于招收实验室新成员时所开发的系统，前端采用 React.js + TypeScript + Three.js（用于动效），后端采用 Next.js + Postgres + Prisma.
提供了
![interface](https://www.gitkraken.com/wp-content/uploads/2022/10/netflix-150x150.png)
- 用户端注册登录，上传简历填写表单，查看简历状态
- 消息队列推送简历状态更新
- 后台的审核和操作
